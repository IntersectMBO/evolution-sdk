# model

## KeyHash (class)

KeyHash as a TaggedClass (breaking change from branded hex string).

```
addr_keyhash = hash28
```

Follows CIP-0019 binary representation.

Stores raw 28-byte value for performance.

**Signature**

```ts
export declare class KeyHash
```

Added in v2.0.0
